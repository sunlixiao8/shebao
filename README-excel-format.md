# Excel文件格式要求

## 问题说明

上传失败是因为Excel文件格式不符合要求。请按照以下格式调整：

## ✅ 正确的Excel格式

### Sheet: cities (城市标准)
```
城市名称 | 年份 | 基数下限 | 基数上限 | 缴纳比例
佛山     | 2024 | 4546    | 22941   | 0.152
广州     | 2024 | 5284    | 26421   | 0.155
```

**说明：**
- 第一行必须是标题行（城市名称, 年份, 基数下限, 基数上限, 缴纳比例）
- 从第二行开始是数据
- 列顺序必须严格一致

### Sheet: salaries (员工工资)
```
员工工号 | 员工姓名 | 年月   | 工资金额 | 城市
001      | 张三     | 202401 | 8000     | 佛山
001      | 张三     | 202402 | 8000     | 佛山
002      | 李四     | 202401 | 9000     | 佛山
```

**说明：**
- 第一行必须是标题行
- 年月格式：YYYYMM（如 202401 表示2024年1月）
- 同一员工需要12个月的数据（1-12月）

## 🔧 如何修复

### 方法1：手动修改现有的Excel文件
1. 打开 cities.xlsx
2. 确认列顺序为：城市名称, 年份, 基数下限, 基数上限, 缴纳比例
3. 确认第一行是标题，第二行开始是数据
4. 保存后重新上传

### 方法2：创建新Excel文件
按照上面的格式创建新的Excel文件，确保：
- 有两个sheet：cities 和 salaries
- 列顺序正确
- 标题行存在

## 🧪 验证数据

上传前请检查：
1. Excel文件包含 cities 和 salaries 两个sheet
2. 列顺序与要求一致
3. 数据从第二行开始
4. 数值格式正确（数字不要带单位）
